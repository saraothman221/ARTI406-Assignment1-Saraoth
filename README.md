# ARTI406 - Assignment 1: Exploratory Data Analysis (EDA)

## 1. Dataset Information
- **Dataset Name:** Mall Customers Dataset
- **Source:** [Kaggle - Mall Customers](https://www.kaggle.com/datasets/vjchoudhary7/customer-segmentation-tutorial-in-python)
- **Format:** CSV File

## 2. Dataset Overview
- **Number of Rows:** 200
- **Number of Columns:** 5

## 3. Description of Features (Columns)
| Column Name | Description | Data Type |
|-------------|-------------|-----------|
| **CustomerID** | Unique identifier for each customer | Numerical |
| **Gender** | Gender of the customer (Male/Female) | Categorical |
| **Age** | Age of the customer in years | Numerical |
| **Annual Income (k$)** | Annual income of the customer in thousands of dollars | Numerical |
| **Spending Score (1-100)** | Score assigned by the mall based on customer behavior | Numerical |

## 4. Purpose of Using This Dataset
The primary goal of using this dataset is to perform **Exploratory Data Analysis (EDA)** to understand customer demographics and behavior. By analyzing the relationship between age, income, and spending scores, we aim to identify potential customer segments (clusters) that can help in targeted marketing strategies.
